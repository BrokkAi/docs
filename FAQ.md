## What languages does Brokk support?

You can use Brokk to code in any language, but the languages with full Context Engine support include

- Java
- Scala
- C++, C
- C#
- Rust
- Go
- Python
- JavaScript, JSX
- TypeScript, TSX
- PHP

## Why did Brokk stop working after upgrading from version 0.15?

The 0.15 version of our auto-updating installer had a bug. Grabbing a fresh installer and running that will fix it.
